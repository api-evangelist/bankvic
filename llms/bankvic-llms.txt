# BankVic

> BankVic (Police Financial Services Limited) is an Australian customer-owned mutual bank and authorised deposit-taking institution (ADI), founded 1974 in Melbourne. As a CDR-accredited Data Holder it publishes a public, unauthenticated Consumer Data Right Product Reference Data (PRD) API conforming to the DSB Consumer Data Standards.

## APIs

- [BankVic CDR Product Reference Data API](https://www.bankvic.com.au/get-help/open-banking/): Public PRD API — product catalogue (accounts, loans, term deposits, credit cards) with terms and key features. Base: https://ib.bankvic.com.au/openbanking/cds-au/v1
  - GET /banking/products — list current products (paginated; requires header x-v: 4)
  - GET /banking/products/{productId} — full product detail
